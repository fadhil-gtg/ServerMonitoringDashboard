Process

Cara Kerja Sistem

Alur sederhana dari instalasi hingga monitoring berjalan secara otomatis.

Step 01

Deploy Docker

Instalasi Docker di server Linux Ubuntu dan deploy container Prometheus + Node Exporter.

Step 02

Konfigurasi

Konfigurasi Prometheus scrape target ke Node Exporter dan atur interval pengambilan data.

Step 03

Koneksi Jaringan

NOC mengkonfigurasi jaringan LAN agar dashboard dapat mengakses API Prometheus.

Step 04

Live Monitoring

Dashboard mengambil data dari Prometheus API dan menampilkan grafik real-time setiap 5 detik.
